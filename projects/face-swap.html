Face Swap Application

AI-Powered Face Swapping for Images and Videos

Project Overview

The Face Swap Application is an advanced AI-powered tool that enables seamless face swapping in both images and short videos. Built using cutting-edge Generative AI models including Stable Diffusion and Flux, this project leverages open-source Hugging Face models to deliver high-quality, realistic face transformations.

Technical Details
