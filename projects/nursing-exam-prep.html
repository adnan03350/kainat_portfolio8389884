Nursing Home Administrator Exam Prep API

AI-Powered RAG System for Certification Preparation

Project Overview

The Nursing Home Administrator Exam Prep API is an AI-driven backend system designed to help users prepare for the Nursing Home Administrator certification exam. It leverages a Retrieval-Augmented Generation (RAG) pipeline built with Qdrant vector database and OpenAI GPT models to generate dynamic study materials, lessons, quizzes, exams, and flashcards.

API Endpoints

  • /take-exam - Generates full-length mock exams
  • /flashcards - Builds flashcards for memory reinforcement
  • /health - System health check
  • /ping - Keeps the service active

Learning Methodology

The system uses semantic search via embeddings from sentence-transformers to retrieve the most relevant content, then generates structured responses through GPT-based text generation. This creates a personalized learning experience tailored to each user's needs and knowledge gaps.
