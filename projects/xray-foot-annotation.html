X-Ray Foot Annotation SaaS Platform

AI-Powered Medical Imaging Analysis System

Project Overview

The X-Ray Foot Annotation System is an end-to-end AI-powered SaaS platform developed to automate the classification and annotation of foot X-ray images for orthopedic analysis. The system leverages deep learning (ResNet-18 CNN models) to identify foot view types, detect abnormalities, and generate precise anatomical annotations.

Technologies Used

Python TensorFlow/Keras ResNet-18 OpenCV JSON Annotation Flask/FastAPI SaaS Architecture

Specialized CNN Models

  • View Classification Model: Determines whether an X-ray image is of DP (Dorsoplantar) or Lateral (Lat) view
  • DP View Model: Detects and annotates key anatomical angles including the talus and foot line
  • Lat View Classification Model: Classifies whether the lateral image is normal or shows signs of a heel spur
  • Heel Spur Annotation Model: Automatically highlights the spur region with circular bounding markers and annotates the talus and foot line

Workflow Summary

  • Performed manual annotation and labeling of X-ray datasets for both DP and Lat views
  • Stored annotation data in JSON format linked with each image for structured training input
  • Applied preprocessing and augmentation for robust model generalization
  • Implemented automated inference pipeline to classify, detect, and annotate X-rays based on view and condition
  • Deployed as a SaaS solution, allowing users to upload X-rays and receive annotated results instantly
