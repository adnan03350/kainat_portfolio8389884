JSSL Football League Bot

AI-Powered Football Registration Assistant

Project Overview

The JSSL Football League Bot is a conversational AI assistant designed to streamline free trial registrations, camp sign-ups, and FAQs for the JSSL Football League. Built using Voiceflow, the bot guides users through natural conversation to collect essential information and provides personalized recommendations.

Technologies Used

Voiceflow Twilio SendGrid Email Automation

Key Features

  • Conversational flow for personalized data collection
  • Automated program and venue selection based on user details
  • Twilio WhatsApp and SendGrid email integration for instant confirmations
  • FAQ support to assist users throughout the registration process
  • Fully automated and scalable workflow built with Voiceflow

Impact

This solution enhances user engagement, reduces manual data entry, and provides a smooth, automated registration experience for football players and their parents.

Workflow Description

The bot intelligently recommends the most suitable football programs based on the user's age group and gender, and allows the user to select a preferred venue. Once registration is complete, the bot automatically triggers confirmation messages via WhatsApp and email using Twilio and SendGrid integration.
