Messy Bible AI

AI-Powered Bible Study and Content Generation Platform

Project Overview

The Bible RAG Project is an AI-powered application designed to deliver structured, contextually accurate responses from the King James Version (KJV) of the Bible. The system uses Retrieval-Augmented Generation (RAG) to retrieve relevant passages and generate structured outputs using prompt engineering and OpenAI's GPT models.

Technologies Used

OpenAI GPT Prompt Engineering Python Flask/FastAPI CSV Vector Database Embeddings REST API

Key Functionalities

  • /bible/query - Retrieves relevant verses based on natural language questions
  • /sermon/generate - Generates AI-assisted sermon outlines and key talking points
  • /game/generate - Creates Bible-based quizzes and games dynamically
  • /social/caption, /social/quote, /social/reel - Produces ready-to-use content for social media
  • /verse/mapping - Maps verses to contextual topics or themes
  • /lessons/generate - Builds structured lesson plans from Bible content

Technical Highlights

  • Used OpenAI GPT API for text generation and structured response creation
  • Implemented prompt engineering to ensure consistent JSON-formatted outputs
  • Designed an internal RAG pipeline using CSV-based embeddings for semantic search
  • Built a scalable REST API architecture documented via Swagger (OpenAPI 3.1)
  • Enabled multi-module support for social media, sermons, and lesson generation
