Job Assistant Automation System

AI-Powered Job Search & Recruitment Automation

Project Overview

The Job Assistant Automation System is an AI-powered workflow designed to streamline job data collection, analysis, and communication for recruiters and job seekers. The system intelligently integrates multiple tools to automate the full lead generation and communication cycle — from data sourcing to personalized outreach.

Technologies Used

n8n Telegram Bot Google Sheets OpenAI API HTTP Requests Gmail Google Drive

Key Features

  • Telegram bot for real-time job queries and updates
  • Integration with Google Sheets for dynamic record management
  • OpenAI-powered job parsing and data extraction
  • Automated email generation and file sharing via Gmail and Google Drive
  • Smart logic branches to filter valid leads and append them automatically
  • Fully modular and scalable automation built inside n8n

Workflow Process

The process starts with a Google Sheet that stores job listings and applicant details. The automation fetches this data, processes it through an OpenAI-based agent, and intelligently extracts key information such as company names, contact emails, and job roles.

Once validated, the data is sent through an automated pipeline that updates records, sends follow-up messages on Telegram, and emails relevant documents through Gmail. The Telegram bot acts as the interactive front-end for recruiters, allowing them to trigger searches, check job statuses, and receive summarized job reports in real time.

Business Outcome

This project eliminates repetitive tasks like manual data entry, lead lookup, and job follow-ups, transforming them into an efficient, intelligent, and automated process. It enables faster data handling, instant communication, and smarter decision-making — all through a single connected automation workflow.

AI Integration

The system leverages OpenAI's capabilities to intelligently parse job descriptions, extract relevant information, and generate personalized communication, significantly reducing manual effort while improving accuracy and efficiency in the recruitment process.
